# Guía para Entregar tu Actividad Práctica

¡Hola Julio! He creado una Landing Page muy llamativa (estilo Glassmorphism con un fondo interactivo de partículas) para que destaques como **"Creative Developer & Innovator"**.

Aquí tienes los pasos exactos para cumplir con todos los entregables de tu actividad:

## 1. Clonar y Ejecutar de Forma Local
El proyecto ya está creado en tu carpeta:
`c:\Users\Admin\Downloads\JULIOTRABAJOZODOXODDOX[\portfolio-julio`

- **Para ejecutarlo:** Simplemente haz doble clic en el archivo `index.html` que está dentro de la carpeta `portfolio-julio`. Se abrirá en tu navegador (Chrome, Edge, etc.).
- **Captura de Pantalla:** Una vez abierto y viendo el increíble diseño con partículas, toma la captura de pantalla requerida para tu entrega.

## 2. Subir el Proyecto a GitHub (Enlace al Repositorio)
Ya he inicializado Git y hecho el primer commit por ti. Para subirlo a GitHub, sigue estos pasos:

1. Ve a [GitHub](https://github.com/) e inicia sesión.
2. Crea un nuevo repositorio (Create a new repository) llamado `portfolio-julio`. No marques la opción de inicializar con un README (déjalo vacío).
3. Copia la URL de tu nuevo repositorio (ej. `https://github.com/tu-usuario/portfolio-julio.git`).
4. Abre la terminal o consola de comandos, asegúrate de estar dentro de la carpeta `portfolio-julio` y ejecuta esto (reemplazando la URL):

```bash
git remote add origin URL_DE_TU_REPOSITORIO
git branch -M main
git push -u origin main
```
Con esto ya tendrás tu enlace al repositorio de GitHub para la entrega.

## 3. Desplegar en Netlify o Vercel
La forma más fácil de desplegar un sitio HTML/CSS/JS moderno:

**Con Netlify:**
1. Ve a [Netlify](https://www.netlify.com/) e inicia sesión con tu cuenta de GitHub.
2. Haz clic en **Add new site** -> **Import an existing project**.
3. Selecciona **GitHub** y autoriza la conexión.
4. Elige tu repositorio `portfolio-julio`.
5. Deja las configuraciones por defecto (Deploy settings) y haz clic en **Deploy site**.
6. ¡Listo! En segundos Netlify te dará un enlace público (ej. `tu-sitio.netlify.app`), que será el que debas entregar.

¡Éxitos con tu entrega! El diseño que hemos creado definitivamente destacará frente al resto.
